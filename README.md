# Syntecxhub E-Commerce MERN

A complete full-stack **MERN E-Commerce website** featuring modern UI, secure authentication, admin dashboard, product management, Cloudinary image uploads, cart + checkout system, and **Stripe payment integration**.

---

## 🚀 Features

### 🛍 User Features
- Register & login with JWT authentication  
- Browse products & categories  
- Product detail page  
- Add to cart / update cart / remove items  
- Checkout with **Stripe Payments**  
- View order history  
- Manage profile & address  

### 🔐 Admin Features
- Admin login  
- Create / Edit / Delete products  
- Upload product images (Cloudinary)  
- Manage categories  
- Manage orders (pending → shipped → delivered)  
- Manage users  

### ⚙️ Technical Features
- REST API using Express  
- MongoDB + Mongoose models  
- Stripe PaymentIntent API  
- Cloudinary image upload support  
- Protected routes (auth + admin middleware)  
- TailwindCSS + optional DaisyUI styling  
- Vite-based React frontend (fast build + dev server)

---

## 🧱 Tech Stack

### Frontend  
- React + Vite  
- React Router  
- Axios  
- TailwindCSS (DaisyUI optional)  
- Stripe React SDK  

### Backend  
- Node.js  
- Express  
- MongoDB + Mongoose  
- JSON Web Token (JWT)  
- Cloudinary  
- Stripe Node SDK  
- Multer (file upload middleware)
